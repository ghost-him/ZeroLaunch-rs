# 代码签名策略

## 📜 认证信息

ZeroLaunch-rs 的所有发布版本均经过专业的代码签名认证，确保软件的完整性和安全性。

### 签名机构
- **证书颁发**：[SignPath Foundation](https://signpath.org)
- **签名服务**：[SignPath.io](https://signpath.io)

### 团队角色
- **提交者与审核者**：[ghost-him](https://github.com/ghost-him)
- **批准者**：[ghost-him](https://github.com/ghost-him)

## 🔒 代码签名的意义

代码签名提供了以下保障：
1. **来源可信**：证明软件由可信的开发者发布
2. **完整性验证**：确保软件在传输过程中未被篡改
3. **Windows SmartScreen**：减少 Windows 安全警告
4. **安装信任**：提升用户安装时的信任度

## 💝 致谢

感谢 [SignPath.io](https://signpath.io) 为开源项目提供的免费代码签名服务，这对保障用户安全和提升软件可信度至关重要。
